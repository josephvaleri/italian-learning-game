# 🇮🇹 Italian Learning Game - Desktop Launchers

Quick shortcuts to launch the Italian Learning Game in your browser!

## 🚀 How to Use

### **Windows Users**
1. Download: `Launch_Italian_Game.bat`
2. Double-click the file
3. Game opens in your default browser!

### **Mac Users**
1. Download: `Launch_Italian_Game.command`
2. **First time only:** Right-click → "Open" → Click "Open" to allow
3. After first time: Just double-click to launch!

### **Any Computer (Universal)**
1. Download: `Italian_Learning_Game.url`
2. Double-click to open in browser
3. Works on Windows, Mac, and Linux!

## 🔄 Updating the Game

The game reads from the Excel file hosted on GitHub. To update:
1. Edit `Italian_Phrases_and_Vocab-for_game_April_2026.xlsx` on your computer
2. Upload to GitHub repository: https://github.com/josephvaleri/italian-learning-game
3. Changes appear immediately (may need to refresh browser)
